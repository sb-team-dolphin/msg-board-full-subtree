# Terraform 실행 파일 폴더

이 폴더에 `terraform.exe` 파일을 추가해야 합니다.

## 다운로드 방법

1. [Terraform 공식 다운로드 페이지](https://www.terraform.io/downloads)로 이동
2. Windows 버전 선택
3. ZIP 파일 다운로드
4. `terraform.exe` 파일을 이 폴더(`bin/`)에 복사

## 파일 구조

```
bin/
└─ terraform.exe  ← 여기에 추가!
```

## 라이선스

Terraform은 HashiCorp BUSL-1.1 라이선스를 따릅니다.
이 애플리케이션은 비상업적 용도로만 사용 가능합니다.

자세한 내용은 [TERRAFORM_LICENSE](../TERRAFORM_LICENSE) 파일을 참조하세요.
