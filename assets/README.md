# Assets 폴더

이 폴더에는 애플리케이션 아이콘 및 이미지 파일이 포함됩니다.

## 필요한 파일

### 아이콘 파일
- `icon.png` - 앱 창 아이콘 (256x256 이상 권장)
- `icon.ico` - Windows 실행 파일 아이콘

## 아이콘 생성 방법

### 온라인 도구 사용
1. [Favicon Generator](https://favicon.io/) 방문
2. 원하는 이모지 또는 텍스트로 아이콘 생성
3. PNG와 ICO 파일 다운로드
4. 이 폴더에 복사

### 수동 생성
1. 256x256 크기의 PNG 이미지 준비
2. [ConvertICO](https://convertio.co/png-ico/)에서 ICO로 변환
3. 파일을 이 폴더에 저장

## 임시 사용

개발 중에는 아이콘이 없어도 앱이 정상 작동합니다.
단, 빌드 시에는 아이콘 파일이 필요합니다.
